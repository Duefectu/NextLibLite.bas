**Commands included:**
- *Debug and internal:*
    - NextInit
    - BREAK
    - BBREAK
- *NextReg:*
    - NextReg
    - NextRegA
    - GetReg
- *Layer 2 (256x192 256 colours):*
    - ShowLayer2
    - CLS256
    - PlotL2
    - ScrollLayer
- *Sprites:*
    - InitSprites
    - RemoveSprite
    - UpdateSprite
- *Tiles:*
    - DoTile
    - DoTile8
- *File operations:*
    - LoadSD
    - SaveSD
    - LoadBMP
